# Asif Mogal | Developer Portfolio Website

Welcome to the repository of my personal developer portfolio website! This site is designed to showcase my skills, projects, and academic background as an aspiring software developer.

🔗 **Live Link**: [https://asif-mogal.github.io/](https://asif-mogal.github.io/)

---

## 🎨 Design & Features

This website is styled with a premium **Developer IDE/Terminal Theme** featuring:
- **Responsive Layout**: Adapts seamlessly to smartphones, tablets, and widescreen desktop monitors.
- **Dynamic Theme Mode**: Smooth toggling between Light and Dark mode using custom CSS variables (default: Dark Mode).
- **Floating Background Glows**: Slow-drifting blurred radial gradients that provide a state-of-the-art glassmorphic depth effect.
- **Terminal Simulator**: Typewriter coding animations in the Hero section reflecting my development domains.
- **IDE Project Tabs**: Interactive terminal widgets for **Shopsphere** and **MockMate** that allow users to select tabs (`Overview`, `Features`, `Tech Stack`, `Setup`) directly in the UI.
- **Functional Contact Form**: Direct AJAX emailing powered by **Web3Forms** (with Formspree and local `mailto:` fallbacks).

---

## 🛠️ Technology Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Variables, Flexbox, Grid, Keyframe Animations)
- **Logic**: Vanilla JavaScript (ES6+, Intersection Observer API)
- **Icons**: FontAwesome v6
- **Contact Backend**: Web3Forms API
- **Deployment**: GitHub Pages (SSL secured)

## 🚀 Local Run & Setup

Since this is a static website, there are **no installation dependencies or build steps** required.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Asif-Mogal/Asif-Mogal.github.io.git
   cd Asif-Mogal.github.io
   ```
2. **Open the website**:
   - Double-click **`index.html`** in your file manager to open it directly in your web browser.
   - Alternatively, open the directory in VS Code and run it using the **Live Server** extension.
