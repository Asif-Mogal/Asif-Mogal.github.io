// SyncBoard — Collaborative Whiteboard

SyncBoard is a premium, low-latency, real-time collaborative drawing whiteboard application. Multiple users can join a shared session using a 6-digit room code, draw freehand and vector shapes, view live mouse cursors labeled with usernames, undo stroke transactions, and persist their workspaces permanently.

Backend: Spring Boot 3.3, Java 21
Real-time: WebSockets, STOMP, Redis
Frontend: React 19, HTML5 Canvas
DB Type: PostgreSQL

Click the tabs above to explore features, specific technologies used, and CLI command instructions to launch locally.

  • > Real-Time Collaboration: Dynamic synchronizations of drawing strokes and shape vectors powered by Spring Boot WebSockets and Redis Pub/Sub.
  • > Active Cursors Overlay: Floating, custom-colored indicators showing peer cursors in real-time, throttled to 35ms to optimize network bandwidth.
  • > High-DPI Canvas Rendering: Scaled drawing context mapping to logical 1920x1080 dimensions using the browser's devicePixelRatio to prevent blurry lines on Retina/4K screens.
  • > Smart Shape Fills: Supports Pen (freehand), Line, Rectangle, and Circle vector tools. Rectangles and circles feature a premium 25% transparent fill toggle matching the stroke color.
  • > Transactional Undo: A single click on the Undo button erases the entire line drawn between pressing the mouse down and releasing it (rather than tiny segment pixels).
  • > Session Management: Active users sidebar. A SessionDisconnectEvent listener automatically clears active users in Redis if they close the tab.
  • > Workspace Exporter & Saving: Export clean, high-resolution .png downloads with slate-900 backgrounds, or commit transient workspaces to PostgreSQL.
Backend & WS:
Java 21 Spring Boot 3.3 Spring WebSocket (STOMP) SockJS
Caching & DB:
Redis (Pub/Sub) Embedded Redis Server PostgreSQL Spring Data JPA
Frontend:
React 19 Vite 5 HTML5 Canvas API Lucide React Vanilla CSS
